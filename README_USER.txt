              Quick Start Guide

FIRST TIME SETUP (One time only):
─────────────────────────────────
1. Double-click "INSTALL.bat"
2. Wait for installation to complete
3. Run: cd backend && npm run seed

DAILY USE:
──────────
Just double-click "ColdDrink-Billing.bat"

That's it! The application will open automatically.


LOGIN CREDENTIALS:
──────────────────
Admin:   admin / admin123
Cashier: cashier / cashier123


ACCESS FROM MOBILE/TABLET:
──────────────────────────
1. Make sure phone is on same WiFi as computer
2. Find computer IP: Open CMD, type "ipconfig"
3. On phone browser: http://YOUR-IP:3000
   Example: http://192.168.1.100:3000


WHERE IS MY DATA STORED?
────────────────────────
All your billing data is saved locally at:
C:\Users\YOUR-NAME\ColdDrinkBilling\database

Your data is 100% on YOUR computer - no cloud, no server fees!


TO CLOSE APPLICATION:
─────────────────────
Close the black command window or press any key in it.
